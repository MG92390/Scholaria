# Assets Temporaires créés

Les fichiers suivants ont été créés comme placeholders :

- icon.png (1024x1024) - Icône de l'app
- adaptive-icon.png (1024x1024) - Icône adaptative Android
- splash.png (1242x2436) - Écran de démarrage
- favicon.png (48x48) - Favicon web

Ces fichiers sont temporaires et doivent être remplacés par vos vrais assets graphiques.

## Recommandations pour vos assets finaux :

### Icône (icon.png)

- Taille : 1024x1024px
- Format : PNG
- Pas de transparence sur les bords
- Design simple et reconnaissable

### Icône adaptative Android (adaptive-icon.png)

- Taille : 1024x1024px
- Zone de sécurité : cercle de 768px centré
- Le système Android peut découper en différentes formes

### Splash Screen (splash.png)

- Taille : 1242x2436px (iPhone X)
- Format : PNG
- Couleur de fond : #1e40af (bleu Scholaria)
- Logo centré

### Favicon (favicon.png)

- Taille : 48x48px
- Version miniature de votre icône
- Bien visible à petite taille
